# Math Statistic 2026 Repository

This repository contains the original exam source PDFs and the web training app.

## App location

The exam-training website is in:

`/Users/nicolas_stpierre/Documents/Math statistic 2026/trainer`

## Run the website

```bash
cd trainer
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Public website

The trainer is configured for GitHub Pages auto-deploy from `main`:

- Expected URL: [https://stni25xr.github.io/math-statistic-2026/](https://stni25xr.github.io/math-statistic-2026/)
- Workflow: `.github/workflows/deploy-pages.yml`
